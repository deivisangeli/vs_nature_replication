# Replication code for "Do Virtue Signals Signal Virtue?"
This repo was prepared to accompany our submission to Nature. The anonymized data include allowes the reader to replicate all our tables and figures.

# Requirements
Do-files were run in Stata 17. The following additional packages need to be installed:
- blindschemes
- estout
- coefplot
- reghdfe
- listtab
- labutil
- distinct
- parmest
